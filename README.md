# 2x2 bred
[![Modrinth Downloads](https://img.shields.io/modrinth/dt/tbt-bred?style=for-the-badge&label=Modrinth%20Downloads&color=%23188346)](https://modrinth.com/mod/tbt-bred)

an ultra simple datapack for minecraft 1.21.9 and newer that lets you craft bread in a 2x2 grid

this is mostly made for personal use, so i will only backport it to older versions if this actually starts getting downloads for whatever reason 
\
\
if you wanna try to backport (or update if there's a new syntax change) it though, feel free to open a pr!

## crafting
place three wheat in any three separate slots of the crafting menu, like this: \
\
![picture of one of the possible bread crafts; L shaped](https://cdn.modrinth.com/data/OoocHRY7/images/a4b32a04270434c007dc3956056280a2bb6d9d0e.png)
\
\
the craft is shapeless, which means the placement of wheat doesn't matter

## building from source
literally just zip the source and make the file extension .jar, done
```
zip -r ../2x2bred.jar .
 ```
